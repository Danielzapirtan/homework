Calculul ultimei cifre

Problemă: Fie b = 5722010 + 7232010 + 6542010 + 2392010.

Să se calculeze u(b) - ultima cifră a lui b.

Notă: u(xn) = u(u(x)n) - ultima cifră depinde doar de ultima cifră a bazei

Calculăm u(b) pas cu pas:

= u(u(5722010) + u(7232010) + u(6542010) + u(2392010))
= u(u(22010) + u(32010) + u(42010) + u(92010))
= u(u(22010) + u(32010) + u(42010) + u(92010))

Tiparul ultimelor cifre pentru puteri:

Pentru 2: 2¹=2, 2²=4, 2³=8, 2⁴=16→6, 2⁵=32→2, 2⁶=64→4... Se repetă: 2,4,8,6

Pentru 3: 3¹=3, 3²=9, 3³=27→7, 3⁴=81→1, 3⁵=243→3... Se repetă: 3,9,7,1

Pentru 4: 4¹=4, 4²=16→6, 4³=64→4, 4⁴=256→6... Se repetă: 4,6

Pentru 9: 9¹=9, 9²=81→1, 9³=729→9, 9⁴=6561→1... Se repetă: 9,1

= u(u(22) + u(32010) + u(42010) + u(92010))
= u(u(22) + u(32) + u(42010) + u(92010))
= u(u(22) + u(32) + u(42) + u(92010))
= u(u(22) + u(32) + u(42) + u(92))
= u(4 + 9 + 6 + 1)
= u(20)
= 0
u(b) = 0

Explicație detaliată:

Pentru a calcula ultima cifră a unei sume de puteri mari, folosim proprietatea că u(xn) = u(u(x)n).

Aceasta înseamnă că doar ultima cifră a bazei contează pentru calculul ultimei cifre a puterii.

Observăm că ultimele cifre ale puterilor se repetă ciclic:

  • Pentru cifra 2: [2, 4, 8, 6] (se repetă la fiecare 4 puteri)
  • Pentru cifra 3: [3, 9, 7, 1] (se repetă la fiecare 4 puteri)
  • Pentru cifra 4: [4, 6] (se repetă la fiecare 2 puteri)
  • Pentru cifra 9: [9, 1] (se repetă la fiecare 2 puteri)

Pentru a afla poziția în ciclu, grupăm puterile:

  • Pentru 22010: 2010 = 4×502 + 2, deci este aceeași cu a 2-a din grup: 4
  • Pentru 32010: 2010 = 4×502 + 2, deci este aceeași cu a 2-a din grup: 9
  • Pentru 42010: 2010 este par, deci este aceeași cu a 2-a din grup: 6
  • Pentru 92010: 2010 este par, deci este aceeași cu a 2-a din grup: 1

Suma acestora este 4 + 9 + 6 + 1 = 20, iar ultima cifră a sumei este 0.
